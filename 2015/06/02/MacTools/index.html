我在Mac上使用的工具

发表于 2015-06-02   |   分类于 Mac   |     |   阅读次数

工欲善其事，必先利其器。一款好的工具，能节约我们很多的时间，能不浪费时间就不浪费，人生苦短，能把多出来的时间来享受生命，何乐而不为呢！

生活相关

  • 谷歌浏览器
  • Movist播放器
  • 射手播放器一款来自中国的视频播放器，本来可以在线加载好多字幕的。但是被封了。
  • iBooks一款苹果自家出的阅读器，配合epub格式的电子档文件，阅读起来棒极了。
  • 网易云音乐来自网易的音乐播放器，喜欢它的内容。
  • Calibre一款开源的电子书阅读器，支持多种格式。
  • 爱壁纸一款跨平台的壁纸软件，有很多精美的壁纸。
  • Parallels DesktopMac上最优秀的虚拟机软件。
  • GIFBrewery一款就视频转化为gif动画的小软件。
  • RAR解压利器Mac上解压rar格式的文件就靠它了。
  • Paragon NTFS for mac一款可以将文件写到NTFS格式硬盘的软件。有了它就不用把你的硬盘格式化了。
  • Tunnelblick翻墙软件，你懂的。使用非常简单，到这里下载配置文件，双击打开你就可以看外面的风景了。
  • CleanMyMac一款清理软件。
  • ReaderRSS阅读器。
  • licecap一款相当不错的gif录制软件。

图像相关

  • PhotoshopPS神器，不必多说。
  • Gimp一款开源的类似Photoshop的图像处理软件。
  • Sketch设计APP必备神器。
  • Pixelmator不想用Photoshop就用它吧。

编辑器

  • Xcode我认为它才是编辑神器，至少在编写iOS程序上面。
  • Sublime写代码非常的丝滑，结合插件要媲美IDE了。
  • Eclipse配合安卓的ADT来开发安卓软件。
  • Android Studio谷歌自家出的安卓开发软件。
  • MacDown一款用markdown语法写作的编辑器。个人认为是Mac平台上最优秀的markdown编辑器。
  • GitBook Editor一款写GitBook的编辑器。

开发相关

  • Reveal一款配合Xcode使用的界面调试工具，非常的强大。
  • PaintCode一款将设计转化为代码的神器。
  • Prepo一款制作icon的不错的软件。
  • Icon 3x free另一款制作icon的软件。
  • Appicon and Launchimage Maker Lite一款制作icon和iOS设备启动图片的软件。
  • ImageOptim一款压缩图片的软件。
  • GitHubGitHub的客户端。
  • Jason一款解析Json格式文档的小工具，能把Json格式的文档以友好的方式展现出来。
  • MindNode Pro一款思维导图软件，非常的好用。
  • XMind另一款优秀的思维导图软件。
  • Dash文档阅读器，里面有各种语言的文档，win和linux下类似的是zeal。
  • Charles抓包就靠它了。

其他

  • 一个下载Mac软件的好去处，点击这里。
  • 大家有更好的工具请留言给我。
