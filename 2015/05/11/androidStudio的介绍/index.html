初识Android Studio

发表于 2015-05-11   |   分类于 Android   |     |   阅读次数

Android Studio是谷歌推出的Android应用程序开发IDE，虽然现在主流还是用Eclipse来开发Android程序，但是我估计随着谷歌对Android的掌控，Android Studio会成为Android程序开发的主要工具。下面先来看看它的庐山真面目吧！


是不是美美哒？既然Android是源于linux,我就选择一个linux系统作为开发系统环境吧！我选用的系统是deepin 2014.3，安装Android Studio的方法是用Unubtu Make,如果不了解Unubtu Make，请自行google，下面我讲一讲大概的安装方法。

1.添加ppa

sudo add-apt-repository ppa:ubuntu-desktop/ubuntu-make
sudo apt-get update

2.安装Ubuntu Make

sudo apt-get install ubuntu-make

3.安装Android Studio

umake android

如果一切顺利，你就可以到启动器(程序列表)中打开你安装好的Android Studio了！就会像下面那样开启你的Android Studio之旅了。但是打开以后你会遇到一些疑问。


1.第一次打开会出现如下界面，这是在下载SDK的相关组件。


解决办法：耐心等待，不要以为是卡主了，我也等了好久才下载好。当然如果过分就了，就得考虑翻墙了。

２．打开以后出现一下提示


解决办法：这是由于linux上默认安装的java环境是openjdk,所以要按照提示安装一个oracle的java环境。下面是我安装oracle的java环境办法：

ssudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java8-installer
sudo apt-get install oracle-java8-set-default

3.模拟器老是弹出一个说谷歌服务停止的提示框


解决办法：在模拟器中找到Settings–>Apps–>ALL,找到Google Play Services,点击进去，DISABLED就OK了。
